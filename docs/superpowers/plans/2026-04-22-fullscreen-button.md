# Fullscreen Button Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a taped-note fullscreen toggle button to the bottom-right of the case board (Lobby + Board) that lets the host enter/exit browser fullscreen.

**Architecture:** A new `FullscreenButton` component uses the `useFullscreen` hook from `react-gameroom@0.10.0` as the source of truth. It's absolutely positioned inside `CaseBoardLayout` (the shared layout for Lobby and Board), mirroring the Krimi wordmark in the top-right. Visually it reuses the existing `TapedNoteButton` with `variant="icon-button"` and an MUI Fullscreen/FullscreenExit icon.

**Tech Stack:** React 19, TypeScript, MUI 9 (`@mui/icons-material`), `react-gameroom@0.10.0`. No test runner configured — verification is manual via `npm run dev`.

## Task 3: Create the `FullscreenButton` component

**Files:**
- Create: `src/components/board/FullscreenButton.tsx`

This is a stateless component. It calls `useFullscreen()` from `react-gameroom`, returns `null` when unsupported, and otherwise renders a `TapedNoteButton` with an MUI icon inside.

- [ ] **Step 1: Write the component**

Create `src/components/board/FullscreenButton.tsx` with exactly:

```tsx
import FullscreenIcon from '@mui/icons-material/Fullscreen';
import FullscreenExitIcon from '@mui/icons-material/FullscreenExit';
import { useFullscreen } from 'react-gameroom';
import { useI18n } from '../../hooks/useI18n';
import TapedNoteButton from './TapedNoteButton';

export default function FullscreenButton() {
  const { isFullscreen, isSupported, toggle } = useFullscreen();
  const { t } = useI18n();

  if (!isSupported) return null;

  const label = isFullscreen ? t('Exit fullscreen') : t('Enter fullscreen');

  return (
    <TapedNoteButton
      variant="icon-button"
      rotation={-2}
      onClick={toggle}
      aria-label={label}
    >
      {isFullscreen ? <FullscreenExitIcon /> : <FullscreenIcon />}
    </TapedNoteButton>
  );
}
```
